Welcome to GoLink Documentation

GoLink is a powerful tool written in Go for checking broken links in your documentation. This homepage will guide you through all the resources you need to effectively use GoLink

Main Features

  • Internal and external link analysis
  • URL validity check
  • Detailed report generation
  • Ability to ignore certain domains or links
  • CI/CD integration

For getting started, check out our installation guide or follow our tutorial.
