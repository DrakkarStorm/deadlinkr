Bienvenue sur la documentation de GoLink

GoLink est un outil puissant écrit en Go permettant de vérifier les liens morts dans vos documentations. Cette page d'accueil vous guidera vers toutes les ressources dont vous avez besoin pour utiliser efficacement GoLink.

Fonctionnalités principales

  • Analyse de liens internes et externes
  • Vérification de la validité des URL
  • Génération de rapports détaillés
  • Possibilité d'ignorer certains domaines ou liens
  • Intégration avec CI/CD

Pour commencer, consultez notre guide d'installation ou suivez notre tutoriel.
