GoLink Installation Guide

This page provides detailed instructions for installing GoLink on your system.

Prerequisites

Before installing GoLink, make sure you have the following:

  • Go 1.18 or higher (see Go download page)
  • Git (see official Git website)
  • An access to our license server (invalid external link)

Installation with Go

go install github.com/youraccount/golink@latest

Installation from source

git clone https://github.com/votrecompte/golink.git
cd golink
go build -o golink main.go

Configuration

After installation, create a configuration file as described in the configuration documentation.

Installation verification

Run the following command to verify your installation:

golink --version
