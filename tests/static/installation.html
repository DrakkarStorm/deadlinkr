Installation de GoLink

Suivez ces instructions pour installer GoLink sur votre système.

Prérequis

Avant d'installer GoLink, assurez-vous d'avoir :

  • Go 1.18 ou supérieur (voir téléchargement de Go)
  • Git (voir site officiel de Git)
  • Un accès à notre serveur de licences (lien externe invalide)

Installation via Go

go install github.com/votrecompte/golink@latest

Installation depuis les sources

git clone https://github.com/votrecompte/golink.git
cd golink
go build -o golink main.go

Configuration

Après l'installation, créez un fichier de configuration comme indiqué dans la documentation de configuration.

Vérification de l'installation

Exécutez la commande suivante pour vérifier votre installation :

golink --version

Pour plus d'informations, consultez les ressources additionnelles (lien externe invalide) ou posez vos questions sur notre forum communautaire (lien interne invalide).

Si vous rencontrez des problèmes, consultez notre FAQ ou contactez-nous.
